Air Canvas using OpenCV via Python

--Hand freee digital drawing canvas. Without using keyboard or mouse we can draw things on laptop.

--It allows us to doodle in the thin air using the gestures captured by the camera with the results transferred to the screen directly and in real-time. There is also a color-palette and the clear all tool in order to erase all the doodles created.

--Libraries Used
	--OpenCV (pip install opencv-python)
	--Numpy (pip install numpy)
	--Collections - Deque
